# Exposure Calculator

Future exposure calculator using flask. Temporary at calc.manuelpm.me


## Deployment

Using [gunicorn](http://gunicorn.org/) and Apahce ProxyPass.

```bash
chmod +x gunicornrun.sh
./gunicornrun.sh
```

Run in port 5000
